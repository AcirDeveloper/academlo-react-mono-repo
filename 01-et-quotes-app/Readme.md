# ¡Bienvenido a mi aplicación de citas aleatorias!

## Descripción

Esta es una aplicación simple creada con React que muestra citas aleatorias del archivo `quotes.json` cada vez que se carga la página o se hace clic en el botón "Siguiente cita". También cambia el color del fondo, las letras y el botón de manera aleatoria cada vez que se muestra una nueva cita.

## Cómo ejecutar la aplicación

1. Clona este repositorio en tu computadora
2. Abre una terminal y navega hasta la carpeta donde clonaste el repositorio
3. Ejecuta el comando `npm install` para instalar todas las dependencias
4. Ejecuta el comando `npm start` para iniciar la aplicación
5. Abre tu navegador y navega a `http://localhost:3000/` para ver la aplicación en acción

## Tecnologías utilizadas

- React
- HTML
- CSS

## Estructura de archivos

- `src/App.js`: componente principal que renderiza la aplicación y maneja el estado de las citas
- `src/components/Quotes.jsx`: componente que muestra la cita en sí misma
- `src/components/Author.jsx`: componente que muestra la cita y el botón para cambiarla
- `src/quotes.json`: archivo que contiene las citas
